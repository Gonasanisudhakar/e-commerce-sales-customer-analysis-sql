 E-Commerce Sales & Customer Analysis using SQL

 📌 Project Overview
This project analyzes an E-commerce dataset using SQL.
The goal is to understand sales performance, customer behavior, and business trends.

🎯 Objectives
 Perform data cleaning.
 Analyze key performance metrics (KPIs)
 Solve real business queries
 Understand customer behavior

🧹 Data Cleaning
 Checked null values in all tables
 Verified duplicate records
 Ensured data consistency

 📊 KPI Analysis
* Total Revenue
* Total Orders
* Total Customers
* Average Order Value


 📈 Business Analysis
* Revenue by Product Category
* Monthly Revenue Trend
* Top Customers by Revenue
* Top Selling Products
* Orders per Customer

 👥 Customer Behavior Analysis
* Repeat vs One-time Customers
* Customers who never ordered
* Customer Lifetime Value
* Customer Segmentation

📊 Key Insights
* Few product categories generate most of the revenue
* Most customers are one-time buyers (low retention)
* A small group of customers contributes high revenue
* Sales show monthly trends
* Some products have high orders but low revenue

 🛠 Tools Used
* MySQL

 📌 Conclusion
This project demonstrates how SQL can be used to clean data, analyze business performance, and understand customer behavior in an E-commerce dataset.
